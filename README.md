\# BinX Tech AI/ML Internship — Week 1



\## Overview

This repository contains the Week 1 deliverable for the BinX Tech AI/ML Internship Program.

Week 1 focuses on setting up the Python data-science environment and verifying the core

scientific libraries used throughout the internship.



\## Objectives

\- Set up a Python virtual environment

\- Install and verify NumPy, Pandas, Matplotlib, and Jupyter

\- Document the environment setup in a reproducible way

\## Setup Instructions

1\. Clone this repository

2\. Create a virtual environment:



python -m venv venv

venv\\Scripts\\activate      # Windows

source venv/bin/activate   # Mac/Linux

3. Install dependencies:

pip install -r requirements.txt
