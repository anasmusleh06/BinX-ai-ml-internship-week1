# Day 1 — Environment Setup & Jupyter Workflow

**Goal:** Set up a reproducible Python environment and get comfortable with the Jupyter Notebook workflow.

**What's inside:**
- Creating and activating a virtual environment (venv)
- Installing NumPy, Pandas, Matplotlib, and Jupyter
- Confirming installed library versions
- Freezing dependencies to `requirements.txt`
- Initializing Git and committing the first notebook

**Tools used:** Python 3.10+, venv, pip, Jupyter Notebook, Git & GitHub
