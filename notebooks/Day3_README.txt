# Day 3 — NumPy: Numerical Computing

**Goal:** Learn to create, manipulate and vectorize operations over NumPy arrays.

**What's inside:**
- Creating arrays (array, zeros, ones, arange, linspace, random)
- Array attributes, indexing and slicing (1D and 2D)
- Vectorized operations and boolean masking
- Broadcasting, with manual verification of results

**Tools used:** NumPy, Jupyter Notebook
