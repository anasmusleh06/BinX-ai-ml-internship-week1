# Day 4 — Pandas: Tabular Data

**Goal:** Load, clean, filter and aggregate real tabular data using Pandas.

**Dataset:** UCI Adult Census Income (from Kaggle)

**What's inside:**
- Loading the dataset and inspecting its shape, columns and dtypes
- Detecting and handling missing values ("?" replaced with NaN, filled with mode)
- Filtering a meaningful subset of the data
- Grouping and aggregating with `groupby`

**Tools used:** Pandas, Jupyter Notebook
